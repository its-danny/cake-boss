Say what now?

Someone help you out recently? Just give 'em cake! They can redeem the cake for any prizes the server has. More info over on the GitHub.

How does it work?

By default, any user can give up to 5 cakes every hour.

To allow only users with 5 or more cakes to give up to 10 cakes every 24 hours:

  • config set requirement-to-give 5
  • config set give-limit 10
  • config set give-limit-hour-reset 24

Don't like cake?

  • config set nickname "Hotdog Boss!"
  • config set cake-emoji üå≠
  • config set cake-name-singular hotdog
